## How to use

### Download source code:

```bash
git clone https://github.com/wipkanban/wipkanban
```

### Install dependences:
```bash
cd wipkanban

npm install
```

### Run application
```bash

npm run server

```
To access enter the address htts://<YOUR_IP_ADDRESS>:3000 in the browser

### Run tests
The tests are divided between client and server

```bash
#client tests
npm run test:client

#server
npm run test:server
```
